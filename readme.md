## 🔍 What Each File Does

### **raw_construction_data.csv**
- Original data with 10 construction projects
- Contains: project names, costs, timelines, safety data, quality issues

### **data_cleaning.py**
- Python script that processes the raw data
- Creates new columns: budget variance, safety scores, quality scores
- Outputs: cleaned_construction_data.csv

### **queries.sql**
- 6 SQL queries for data analysis
- Shows database knowledge
- Examples: budget analysis, supplier performance, risk identification

### **construction_dashboard.pbix**
- Power BI visualization file
- Contains 6 interactive charts
- Shows: budget variances, project status, quality vs safety
